WEB-DEV Dashboard Concept

This project is a work in progress. While it's not quite a version 1.0 release, it's getting close.
The end result will be a helpful tool for debugging and development.
The list below shows some of the features that are implemented, and/or are in progress:

  • Responsive Website Debugging
  • Page Link Checker
  • Live CSS Stats
  • Styleguide Generator
  • A11Y Compatibility
